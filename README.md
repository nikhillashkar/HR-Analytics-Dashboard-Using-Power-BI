# HR Analytics Dashboard - Power BI Project

## Overview

This project focuses on analyzing HR data using Power BI to generate meaningful insights about employee performance, attrition, workforce distribution, and overall HR metrics. The goal of the project is to transform raw HR data into an interactive KPI dashboard that helps organizations make data-driven decisions.

---

## Dataset

The dataset used in this project contains employee-related information such as:

* Employee ID
* Department
* Job Role
* Gender
* Age
* Salary
* Attrition
* Years at Company
* Performance Rating
* Education Level

The dataset was imported into Power BI for analysis and visualization.

---

## Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation
* **Microsoft Excel / CSV** – Dataset source
* **DAX (Data Analysis Expressions)** – KPI calculations and measures

---

## Project Steps

### 1. Data Loading

* Imported HR dataset into Power BI
* Verified data types and column structure

### 2. Data Cleaning

* Removed duplicate records
* Handled missing/null values
* Standardized column names
* Corrected inconsistent data entries

### 3. Data Transformation

* Created calculated columns and measures using DAX
* Filtered unnecessary fields
* Structured data for better reporting

### 4. Dashboard Development

Built an interactive HR analytics dashboard including:

* Employee Count
* Attrition Rate
* Average Salary
* Department-wise Analysis
* Gender Distribution
* Employee Performance Metrics
* Age Group Analysis

---

## Dashboard Features

* Interactive filters and slicers
* KPI cards for quick insights
* Department-wise visual analysis
* Attrition trend visualization
* Clean and user-friendly interface

---

## Results & Insights

The dashboard helps identify:

* Departments with high attrition
* Employee distribution across job roles
* Workforce diversity metrics
* Key HR performance indicators
* Trends affecting employee retention

---

## How to Run the Project

1. Download the project files
2. Open the `.pbix` file in Power BI Desktop
3. Load the dataset if required
4. Refresh the dashboard to view updated insights

---

## Conclusion

This HR Analytics project demonstrates how Power BI can be used to clean, analyze, and visualize HR data effectively. The dashboard provides actionable insights that support better workforce management and strategic decision-making.
